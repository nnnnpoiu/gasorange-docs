# GasOrange API 文档

GasOrange 大模型中转平台 API 文档

## 快速接入

GasOrange 提供统一的 API 接口，支持文本、图片、视频等多种大模型能力。

### 基础信息

| 配置项 | 值 |
|--------|-----|
| Base URL | `https://www.gasorange.com` |
| 认证方式 | Bearer Token (API Key) |

### 支持的模型类型

- **文本模型** - 大语言模型对话
- **图片模型** - 图像生成
- **视频模型** - Seedance 视频生成

请查看 [快速开始](./kaishi) 了解如何接入 GasOrange API。
